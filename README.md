# 2025-2026-Rollercoaster-SRCS
Repository for the Safety Related Control System for REC 2025-2026 Competition 

This repository serves the purpose of tracking all of the design decisions made as well as their supporting softwares and implementations of the code. 
This includes but is not limited to logical flow, wiring diagrams, block designs, constraints files, bitstreams, C code, and arduino code.
